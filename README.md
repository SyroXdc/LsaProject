# 🛒 L.S.A Crocs - Modern E-Commerce Marketplace

A fast, responsive, and fully functional e-commerce storefront designed for modern lifestyle brands. This project was built to provide a seamless shopping experience with features specifically tailored for the Southeast Asian market.

### 🌐 Live Demo
**[👉 Click here to experience the live website](https://lsa-store.vercel.app/)**

---

## ✨ Key Features
* **WhatsApp Integration:** Custom login and registration system using WhatsApp numbers, maximizing user conversion for local markets.
* **Dynamic Shopping Cart:** Real-time cart updates and seamless checkout flow.
* **Modern UI/UX:** Clean, mobile-first design with categorized product grids and featured collections.
* **Customer Support Ready:** Integrated floating chat and direct links to social media/WhatsApp for immediate customer interaction.
* **Admin Dashboard Access:** Built-in routing for store management (backend integration ready).
